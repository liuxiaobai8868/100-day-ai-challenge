## 关于这个项目 | About

**中文**

我是刘小白。小白不是艺名，是真实状态。

带孩子5年，没有工作，没有代码基础。但我做了一个决定：100天，每天从零搭一款AI应用，全程记录，不剪不藏——包括失败。

这个仓库将记录全部100个项目的代码、思路和踩坑过程。。

**English**

I'm Xiaobai — which literally means "total beginner" in Chinese. The name fits.

5 years as a stay-at-home mom. Zero code. Zero tech background. One decision: build one AI app every day for 100 days and document everything — including the failures.

This repo will document all 100 projects — code, process, and lessons learned.
